## install and usage:

**Termux:**
* `pkg install python2`
* `pkg install git`
* `git clone https://github.com/Avenge-hacker/Firecrack`
* `cd Firecrack`
* `pip2 install -r requirements.txt`
* `python2 firecrack.py`
* `help`

**Linux:**
* `apt-get install python`
* `apt-get install python-pip`
* `apt-get install git`
* `git clone https://github.com/Avenge-hacker/Firecrack`
* `cd Firecrack`
* `pip install -r requirements.txt`
* `python firecrack.py`
* `help`
